1
余事象を用いて確率を求める利点について説明し、サイコロを2回投げて少なくとも1回6が出る確率を余事象を使って求めてください。
解答のポイント ▼
余事象の利点：「少なくとも1回」のような条件では直接計算が複雑になるが、余事象なら簡潔に計算できる。
計算：P(少なくとも1回6) = 1 - P(2回とも6以外) = 1 - (5/6)² = 1 - 25/36 = 11/36
2
等比数列の一般項と和の公式を導出し、初項2、公比3の等比数列の第5項までの和を求めてください。
解答のポイント ▼
一般項：aₙ = a₁ × r^(n-1)
和の公式：Sₙ = a₁(r^n - 1)/(r - 1) (r≠1)
計算：S₅ = 2(3⁵ - 1)/(3 - 1) = 2(243 - 1)/2 = 242
3
定積分∫₀² x²dx を定義に基づいて求め、微分積分学の基本定理との関係を説明してください。
解答のポイント ▼
リーマン和の極限として定義し、区分求積法を使用。
基本定理：F'(x) = x²となるF(x) = x³/3を用いて、[x³/3]₀² = 8/3
4
ベクトルの内積の定義と性質を説明し、なぜ内積が0のとき2つのベクトルが垂直であるといえるのか証明してください。
解答のポイント ▼
内積の定義：a·b = |a||b|cosθ
a·b = 0 ⇔ cosθ = 0 ⇔ θ = 90° を示す
5
三角形の五心（重心、外心、内心、垂心、傍心）のうち重心について、その性質と座標での表し方を説明してください。
解答のポイント ▼
重心は3つの中線の交点で、各頂点から2:1に内分する点。
座標：G = ((x₁+x₂+x₃)/3, (y₁+y₂+y₃)/3)
6
組合せnCrと順列nPrの違いを説明し、7人から3人の委員を選ぶ場合と、7人から会長・副会長・書記を選ぶ場合の数をそれぞれ求めてください。
解答のポイント ▼
組合せは順序を考慮しない、順列は順序を考慮する。
委員：₇C₃ = 35通り、役職付き：₇P₃ = 210通り
7
2次関数y = ax² + bx + cの頂点の座標を平方完成を用いて求める方法を説明し、y = 2x² - 4x + 5の最小値を求めてください。
解答のポイント ▼
平方完成：y = a(x - p)² + q の形に変形
y = 2(x - 1)² + 3、頂点(1, 3)、最小値3
8
数列の漸化式aₙ₊₁ = 2aₙ + 1（a₁ = 1）を解き、一般項を求める手順を説明してください。
解答のポイント ▼
特性方程式を用いて、aₙ₊₁ + 1 = 2(aₙ + 1)の形に変形
bₙ = aₙ + 1とおくと、bₙ = 2ⁿ、よってaₙ = 2ⁿ - 1
9
三角関数の加法定理sin(α+β)を証明し、これを用いてsin75°の値を求めてください。
解答のポイント ▼
単位円上の点の回転を用いて証明
sin75° = sin(45°+30°) = sin45°cos30° + cos45°sin30° = (√6+√2)/4
10
条件付き確率P(A|B)の定義を説明し、サイコロを投げて偶数が出たとき、それが4以上である確率を求めてください。
解答のポイント ▼
P(A|B) = P(A∩B)/P(B)
偶数：{2,4,6}、4以上の偶数：{4,6}
P(4以上|偶数) = (2/6)/(3/6) = 2/3
